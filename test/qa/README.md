# QA Artifacts

This folder stores QA outputs generated when the QA orchestrator guideline is executed.

## Required files per run

- `TEST_SCENARIOS_<YYYY-MM-DD>.csv` - scenario matrix (Excel-compatible)
- `DEFECT_LOG_<YYYY-MM-DD>.csv` - defects found (Excel-compatible)
- `QA_REPORT_<YYYY-MM-DD>.md` - execution report and sign-off decision

## Notes

- Keep one set of files per run date.
- Update status after retest instead of deleting historical reports.
